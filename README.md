FluoRender 3
========

This is the restructuring of FluoRender to simplify the program data tree and improve rendering capabilities.

Building
========

Requirements: Git, CMake, CTest

<h4>OSX/Linux</h4>

<code>mkdir build</code>

<code>cd build</code>

<code>cmake ../src</code>

<code>make</code>

Testing
========

<code>make test</code>
